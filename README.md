# NBA Conference Comparison

## Summary

We analyzed NBA franchise team and player data to determine which
conference is better, the Eastern or Western Conference based on
the following criteria metrics:
- Performance
- Awards
- Intangibles


## Installation
**Necessary Libraries:**
- pandas
- matplotlib
- datetime
- nba_api

```
pip install nba_api
```
